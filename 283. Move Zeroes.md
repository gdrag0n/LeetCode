# Solution
find number of zeroes
perform removal
add 0s at the end

```
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
    
        n= nums.count(0)
        for i in range(n):
            nums.remove(0)
            nums.append(0)
```
